J2Objc Manual

J2Objc is a tool to convert Json string to Objective-c model classes. To use this tool, simply paste a json string and name the model class where you want.

  • Taking a json string with a class name

    • Input:

    User{
	"name" : "Jack",
	"age"  : 15
}

    Output:

    class User which contains two string property "name" and "age"

  • Specify your own types and property name by explicit declare them.

    • Input:

    User{
	"name",
	"id"     : userID,
	"gender" : NSString *gender,
	"age"    : int userAge
}

    Output:

    class User with 4 properties: name, userID , gender ,userAge.

    Note: If no property name are specified, json's key is used as property name. To simplify writing, you can also use this format (json's key can be surrounded by " or ' or nothing):

    // simplified writing 
User{
	name,
	userId,
	gender  
	// 'int gender'   is not supported. Use 'gender: int gender' instead 
}
  • Array type

    • To specify an array variable you can use implicit or explict declaration.

    			
User{
	"friends" : [], // define property: NSArray *fiends; 
	"friends2", [	// define property: NSArray *fiends2; 
		{"name":"John"},
		{"name":"Jack"}
	],
	"friends3", myFriends [	// define property: NSArray *myFiends; 
		{"name":"John"},
		{"name":"Jack"}
	],
	"friends4", NSArray<Friend> myFriendlist [// define property: NSArray *myFriendlist which is elements of Friend; 
		{"name":"John"},	//'Friend' class will be generated if 'Friend' hasn't been defined
		{"name":"Jack"}		//If 'Friend' class has no "name" property, "name" will be add to 'Friend'. 
	] 
}
  • Dictionary and Custom type

    • To specify an dictionary variable or your own custom object you can use implicit or explict declaration.

    			
{
	"status" : true, // define property: BOOL status. Since no top object is defined, this is simply ignored. 
	"data"   : {	
		"user" : {},	// define property: NSDictioanry *user; 
		"user2": NSDictionary *user2, // define property: NSDictioanry *user2; 
		"user3": User *user3{	// define property: User *user3, and create class "User" with property "name" and "age"  
			"name",
			"age"
		}
	}
	
}
